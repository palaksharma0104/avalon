# Avalon
#### because your stories deserve a place as magical as you.

> [!NOTE]
> This specific project does not require any environment variables.


## Installation
- Development script:
  ```node
  npm i
  npm run dev
  ```


- Deployment script:

  ```bash
  npm i
  npm run build

  // If you want to view
  npm preview
  ```

- Go to http://localhost:5173 (default Vite port).
